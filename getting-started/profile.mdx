The Sasini App onboarding pipeline is designed to be frictionless for the user while executing complex validation and encryption protocols in the background. This page details the complete registration lifecycle.

## 1. User Interface Workflow

When a new user launches the application, they are routed through the secure onboarding flow.

<Steps>
  <Step title="Initial Launch & Localization">
    The app automatically detects the device's region. The user is presented with the language selection screen (English, Swahili, French, German).
    
    <Frame>
      <img src="/images/language-select.png" alt="Language Selection Screen" />
    </Frame>
  </Step>
  <Step title="Data Ingestion">
    The user inputs their primary KYC (Know Your Customer) details:
    * Full Legal Name
    * Verified Phone Number (OTP verification required)
    * Farm Location (Captured via device GPS API)
    * Primary Produce (Tea, Coffee, Macadamia, Avocado)
  </Step>
</Steps>

## 2. Backend Architecture: Digital ID Generation

Once the frontend submits the user payload, the Sasini backend initiates the Digital ID generation protocol. This is crucial for maintaining traceability across all deliveries and payments.
